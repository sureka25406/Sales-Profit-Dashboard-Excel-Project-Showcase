📊 Sales & Profit Dashboard – Excel Project


📌 Project Overview

This project showcases an interactive Sales & Profit Dashboard built using Microsoft Excel.
The dashboard provides a comprehensive analysis of business performance by evaluating sales, profit, customer trends, and regional distribution.

The primary objective is to transform raw sales data into meaningful business insights using structured data preparation, Pivot Tables, and interactive visualizations.

🛠 Tools & Technologies

Microsoft Excel

Data Cleaning & Preparation

Excel Tables

Pivot Tables

Pivot Charts

Slicers

Conditional Formatting

📂 Project Workflow
1️⃣ Data Preparation

Imported raw dataset

Cleaned missing and inconsistent values

Formatted data into structured Excel Tables

Created calculated fields for better analysis

2️⃣ Data Analysis

Built multiple Pivot Tables

Generated summary metrics (Sales, Profit, Customer Count)

Performed category-wise and state-wise analysis

3️⃣ Dashboard Development

Designed KPI cards for Total Sales & Profit

Created visual charts for yearly, monthly, and category analysis

Integrated slicers for dynamic filtering

Applied consistent layout and professional formatting

📈 Dashboard Features

🔹 Total Sales & Total Profit Overview

🔹 Profit by Year Analysis

🔹 Sales by Category (Funnel Chart)

🔹 Customer Count Visualization

🔹 Sales by State (Map Chart)

🔹 Top 5 Customers by Profit

🔹 Monthly Sales Trend

🔹 Interactive Filters (Month, Year, Category)
